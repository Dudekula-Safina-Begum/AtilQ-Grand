# 🏨 Atliq Grand – Hotel Revenue & Performance Dashboard

## 📌 Overview
This project focuses on analyzing hotel business performance using an interactive Power BI dashboard.  
The goal is to track key metrics, identify trends, and provide actionable insights to improve revenue and operational efficiency.

---

## 🎯 Objectives
- Monitor overall hotel performance  
- Analyze revenue trends and occupancy patterns  
- Identify top-performing cities, properties, and platforms  
- Detect business challenges like cancellations and low occupancy  
- Provide data-driven recommendations  

---

## 📊 Key Metrics (KPIs)
- Revenue  
- Occupancy %  
- Average Daily Rate (ADR)  
- Revenue Per Available Room (RevPAR)  
- Average Rating  
- Week-over-Week (WoW) Change %  

---

## 📈 Dashboard Features
- Interactive visuals for performance analysis  
- Dynamic KPIs with WoW comparison  
- Slicers for filtering (Week, Month, Room Class, City)  
- Tooltip insights for deeper analysis  
- Bookmark-based popups for detailed views  
- Insights & Recommendations panel  

---

## 🔍 Key Insights
- Bangalore contributes the highest revenue but has slightly lower ratings  
- Weekend occupancy is significantly higher than weekdays  
- Cancellation rate (~25%) is high, impacting revenue realization  
- Majority of bookings come from third-party platforms  
- Some properties outperform others, indicating optimization opportunities  

---

## 💡 Recommendations
- Improve customer experience in high-revenue cities  
- Promote direct bookings to reduce dependency on aggregators  
- Introduce weekday offers to balance occupancy  
- Optimize cancellation policies  
- Focus on improving underperforming properties  
- Increase occupancy to improve RevPAR
